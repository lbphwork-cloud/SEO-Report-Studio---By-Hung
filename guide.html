🔑 Lấy Claude API key trong 5 phút

Để dùng tính năng "Claude AI viết SWOT & Kết luận", bạn cần một API key của riêng mình. Bạn tự trả phí theo lượt dùng — thường rất rẻ.

Vì sao cần key riêng? App này gọi thẳng Claude từ trình duyệt của bạn — key của bạn không bao giờ đi qua server của chúng tôi. Bạn nạp tiền vào tài khoản Anthropic của chính bạn và chỉ bạn trả cho phần mình dùng.
Các bước thực hiện
1

Tạo tài khoản Anthropic

Mở console.anthropic.com → đăng ký bằng email hoặc tài khoản Google. Xác nhận email nếu được yêu cầu.

2

Nạp credit (tiền dùng API)

Vào menu Billing (hoặc Plans & Billing) → Add credits → nạp tối thiểu $5 bằng thẻ Visa/Mastercard.

💡 $5 là đủ tạo rất nhiều báo cáo nếu bạn dùng model Haiku (xem bảng giá bên dưới).

3

Tạo API key

Vào menu API Keys → bấm Create Key → đặt tên tùy ý (vd "SEO Studio") → bấm tạo.

⚠️ Key chỉ hiện một lần duy nhất — copy ngay và lưu lại nơi an toàn.

4

Dán key vào app

Quay lại SEO Report Studio → chọn "Claude AI" → dán chuỗi sk-ant-... vào ô Claude API key. Xong! Key được lưu trong trình duyệt của bạn cho các lần sau.

🔒 Bảo mật: Đừng chia sẻ key cho người khác hay đăng công khai. Nếu lỡ lộ, vào API Keys → xóa key cũ (Revoke) và tạo key mới. Bạn có thể đặt giới hạn chi tiêu (spend limit) trong phần Billing để yên tâm không bị dùng quá tay.
